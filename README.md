# TopUpIn

## Deskripsi
TopUpIn merupakan website layanan top up game berbasis web yang dikembangkan sebagai proyek mata kuliah Startup.

## Kontribusi Saya
Pada proyek ini saya berperan sebagai Frontend Developer dengan kontribusi:
- Halaman Home
- Login
- Register
- Catalog Produk
- Detail Produk
- Checkout
- Riwayat Transaksi

## Teknologi
- PHP
- HTML
- CSS
- JavaScript
- Bootstrap
- MySQL

## Cara Menjalankan
1. Import database `database.sql`.
2. Letakkan project pada folder `htdocs`.
3. Jalankan Apache dan MySQL melalui XAMPP.
4. Akses melalui `http://localhost/topupin_web`.
